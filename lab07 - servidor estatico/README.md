Como acessar o servidor:

1 - Crie uma pasta chamado Servidor

2 - Acesse o prompt de comando
( Precisa aparecer seu C:\alguma coisa)

3 - Adicionar dir nessa pasta para ver tudo o que está dentro da pasta

4 - Adicionar cd na pasta que está a pasta servidor e encontrar ele

5 - Entrar na pasta servidor usando cd

6 - Depois na pasta instalar o npm init

7 - Dentro da pasta vai ter:

    package name: (servidor)
    version: (1.0.0)
    description: este eh meu servidor
    entry point: (index.js) servidor.js
    test command: 
    git repository: 
    keywords: 
    author: Nicolas Chagas
    license: (ISC)
    
8 - Ai depois na pasta instale npm install express --save

9 - Depois instale também o npm install colors

!0 - Para rodar o servidor colocar node servidor.js
